# Movie-Recommendation-System-using-NLP-Streamlit
Built a Streamlit-based movie recommendation system using NLP and TF-IDF vectorization on IMDb data.  Processed movie titles, genres, directors, and cast details with text preprocessing and feature extraction.  Generated personalized movie suggestions by analyzing content similarity and presenting results interactively.
